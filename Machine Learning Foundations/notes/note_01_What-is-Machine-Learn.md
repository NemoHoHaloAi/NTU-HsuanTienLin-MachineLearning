# What is Machine Learn?

## 三个条件
1. 有某种隐藏的规则存在；
2. 规则无法直接写出，或许是未知的，或许是太复杂，或许是太多；
3. 足够多的数据；

## 抽象
- input - X
- output - Y
- data - X[1]~X[n]
- target function - f
- hypothesis - set(g1, g2, g3)

从`data`（数据源）开始，经过`hypothesis`（假设的模型可能，通常有无数个），得到`g`（结果模型），送入`input`（待预测数据），得到`output`（预测结果）；

## 关系
- 数据挖掘
- AI
- 统计

是一门涉及多学科的交叉学科：统计学、概率论、最优化、计算机、信息学、生物学等；
