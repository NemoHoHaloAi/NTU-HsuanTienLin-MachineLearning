# 各式各样的机器学习

## 从output区分
- 二分类：即是否问题，比如判断是否垃圾邮件；
- 多分类：有限的大于两个的类别问题，比如判断癌症的种类；
- 回归：输出为实数，可以是整个实数范围，可以是固定范围内的实数，比如分数为0~100内的实数；
- 结构化学习：输出无法穷举出来，比如对一句话做词性标注后，输出每个词词性组成的字符串，这个结果是有一定结构的，但是无法穷举，因此属于结构化学习；

## 根据label区分
- 监督学习：any Yn
- 非监督学习：no Yn
- 半监督学习semi：some Yn
- 强化学习：通过奖励、惩罚机制来学习
