# 3个ML的准则

知识点：
1. 奥卡姆剃刀：
    1. 越简单的模型越好；
    2. 简单的model H对应简单的h；
    3. simple go first，like linear；
2. 抽样偏差：
    1. 训练数据与实际问题数据来自不同的函数分布；
    2. 验证数据与训练数据划分有问题，没能对齐实际测试环境；
3. 数据窥探：
    1. 6年数据训练、2年数据验证时，做缩放使用了全部8年的均值、标准差等简介对验证数据进行了污染；
    2. 研究中，每个paper都是基于上一个paper的基础上，而考虑模型复杂度时，只考虑了自己，实际上上一个模型对自己的模型是有污染的；
    3. 准则：
        1. 严格不要污染test；
        2. 进行验证；
        3. 没有数据窥探前做feature；
        4. 保持怀疑，尤其是结果特别好时；
